## Features

- Semantic HTML structure
- Responsive layout for mobile, tablet, and desktop
- Accessible avatar with meaningful alt text
- Live current epoch time in milliseconds
- Social links that open in a new tab
- Distinct hobbies and dislikes lists
- Visible keyboard focus styles
- Exact `data-testid` attributes for stable testing

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Run Locally

Open `index.html` in your browser

## Accessibility Notes

- Uses semantic tags like `article`, `figure`, `header`, `nav`, and `section`
- Avatar image includes meaningful alt text
- Links are keyboard-focusable and browser tab opened
- Focus styles are visible
- Time updates use `aria-live="polite"`

## Design Decisions

- Used CSS Grid for responsive layout
- Kept the UI lightweight and readable
- Updated time every 1000ms using `Date.now()`
- Grouped hobbies and dislikes into separate semantic sections
